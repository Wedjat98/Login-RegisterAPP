# login-register-vue-node

## ログイン・登録機能

ログイン・登録機能、フルスタック、データベースへのデータロギング、フロントエンドとバックエンドの分離
------

## プロジェクトの展開

### データベース

データベースのテーブルはvarchar型で、シンプルなユーザーテーブルが作成される

### プロジェクトの初期化
```
npm install
```

### バックエンドの初期化と実行
```
cd service
npm install
node app.js
```

### フロントエンドプロジェクトの実行
```
npm run serve
```
